## 🚀 Hosting on GitHub Pages

1. Create a new repo (e.g. `portfolio`)
2. Upload all files keeping the folder structure above
3. Go to **Settings → Pages → Source → main branch / root**
4. Your portfolio will be live at `https://[your-username].github.io/portfolio/`

> **Note:** `index.html`, `remodel_casestudy.html`, and `remodel_prototype.html` are static and work perfectly on GitHub Pages.  
> The `golden-cookie/` and `kscope/` folders require a PHP + MySQL server (e.g. InfinityFree, 000webhost, or your own hosting) to run the backend features.
